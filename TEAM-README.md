# Project 2 Team Checklist

## Core Requirements

- Node + Express
- Handlebars
- PostgreSQL
- MVC structure
- Full CRUD (`GET`, `POST`, `PUT`, `DELETE`)
- Authentication
- Environment variables
- Render deployment

## Database Minimum

- At least 2 tables
- At least 5 rows per table

Use either:
- Sequelize, or
- `pg` (SQL-first)

## Exploration Requirement

Include at least 2 of:
- External APIs
- New npm packages
- New frontend/CSS tech
- New backend/tooling concepts

## README Requirement

Include:
- Project description
- Technologies used
- Screenshots
- Deployed URL

## Presentation Requirement

Cover:
- Pitch
- Concept
- Build process
- Demo
- Future work

All team members speak at least once.

## Submission

Each member submits:
- Deployed app link
- GitHub repo link
